# 该文件列出了项目的核心依赖项。

# --------------------------------------------------------------------------------------
# !! 关于依赖与冲突的重要说明 !!
#
# 1. PyTorch & CUDA:
#    PyTorch的版本至关重要，必须与您的NVIDIA驱动和CUDA工具包版本相匹配。
#    下面的命令行是一个CUDA 11.8的示例。在安装前，请在终端中运行 `nvcc --version` 来验证您的CUDA版本。
#    然后，访问 https://pytorch.org/get-started/locally/ 获取正确的安装命令。
#
# 2. 潜在冲突 (ultralytics vs. timm):
#    某些版本的 `ultralytics` (用于YOLO) 可能对 `timm` 库有严格的依赖，
#    这可能与Metric3D等其他模型所需的版本冲突。
#    目前的最佳实践是先安装 `ultralytics`，然后再安装其他软件包。
#    如果遇到冲突，作为最后的手段，您可能需要为训练不同类型的模型创建独立的虚拟环境。
# --------------------------------------------------------------------------------------

# 强烈建议首先根据您特定的CUDA版本，按照PyTorch官方网站的指引安装PyTorch。
# 这可以避免许多潜在的依赖冲突问题。
# 请访问: https://pytorch.org/get-started/locally/

# --- 核心机器学习与数据处理库 ---
numpy               # 科学计算基础库，用于多维数组和矩阵运算

matplotlib          # 流行的2D绘图库，用于数据可视化
seaborn             # 基于matplotlib的高级可视化库，提供更美观的统计图形
tqdm                # 一个快速、可扩展的进度条工具，用于可视化循环的执行进度
PyYAML              # 用于读写YAML文件，本项目用作配置文件
scikit-learn        # 机器学习库，本项目主要用于计算评估指标（如混淆矩阵、精确率、召回率等）
